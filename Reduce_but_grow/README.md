# Reduce but grow

8Kyu kata from code wars
https://www.codewars.com/kata/57f780909f7e8e3183000078

## Instructions:
Given a non-empty array of integers, return the result of multiplying the values together in order
Example:
`<[1, 2, 3, 4] => 1 * 2 * 3 * 4 = 24>`
